Welcome to TattingCAD!

Create beautiful tatting patterns with this digital design tool.

Getting Started

1
Add Your First Ring or Chain

Use the left toolbar buttons to add elements. Rings, Chains, SplitRings all at the touch of a button.

2
Write Notation

Select an element and type notation like r: 5ds-p-5ds in the properties panel. The pattern updates automatically according to the stitch count!

3
Add Picots & Joins

Use p for picots, jp for join points, gp for guide points, jpg for guide joined picots.

4
Assign Materials / Colors

Open Options → Materials… to create thread materials with colors or gradients, then assign them to elements from the property bar.

5
Save & Export

Click Save to save your project as a .json file, or export as SVG for printing and sharing.

6
Sharing Your Pattern

Click File → Output notation to clipboard to get all your notation in one place. You can enhance the results by assigning the order of tatting using View → Tatting order and by assigning real world values to the program under Options → Thread Properties.

Notation Syntax

All notation is case-insensitive. Tokens are separated by - or . — both are accepted everywhere.

Element type prefixes

Every notation string must start with one of these prefixes followed by :

r:  Ring
c:  Chain
sr:  Split Ring
jk:  Josephine Knot (parsed as ring, reserved prefix)
fr:  Floating Ring (parsed as ring, reserved prefix)
r: 5ds-p-5ds
Stitch tokens

N is always optional and defaults to 1.

Nds  Double stitch(es) — path length N. ds alone = 1.
Nss  Single stitch(es) — path length N × 0.5
Nlss  Left single stitch(es) — same path size as ss, stored as distinct type
Nrss  Right single stitch(es) — same path size as ss, stored as distinct type
Nrds  Reinforced double stitch(es) — path length N × 2
Picot tokens

Picots do not advance the stitch counter. N before any picot repeats it: 3p = three picots at the same position.

p  Regular picot (medium length)
sp or cp  Small picot
lp  Large picot
jp  Join picot — marks a connection point between elements. Green (free) → yellow (connected) → pink (selected)
jpg  Guide join picot — rendered as a green arm; selectable in join tool and acts as snap point
gp  Guide point — yellow-green diamond on path, snap anchor, no arm rendered
Repeat groups
Nx(...)  Repeat contents N times — 3x(jp-3ds) expands to jp-3ds-jp-3ds-jp-3ds
(...)xN  Same — both orders accepted
r: 3x(jp-3ds)  ← same as jp-3ds-jp-3ds-jp-3ds
r: 5*(3ds-p)  ← * also works as repeat operator
Split Ring
A split ring uses two notation fields — Side A and Side B.
Side A: sr: 5ds-p-5ds
Side B: 3ds-jp-3ds  ← no sr: prefix on Side B
Separators
Use - (dash) or . (dot) — they are identical.
r: 5ds-p-5ds  =  r: 5ds.p.5ds
Pattern export format

When exported, notation is rebuilt from picot data. Join picots include inline connection references:

r: 3ds-jp//r3//-3ds  ← connected to ring 3
r: 3ds-jp//r3//r11//r15//-3ds  ← connected to rings 3, 11 and 15
r: 3ds-jp(?)-3ds  ← join picot with no connection yet
r: 3ds-bjp//r3//-3ds  ← beaded join picot with connection
Worked examples
r: 5ds-p-5ds-p-5ds-p-5ds    Ring, 15 ds, 3 medium picots
r: 3ds-jp-3ds-jp-3ds-jp-3ds  Ring, 12 ds, 3 join picots
r: 3x(jp-3ds)                Same using repeat group
c: 5ds-sp-3ds-lp-3ds-sp-5ds  Chain with small/large/small picots
sr: 5ds-jp-5ds-jp-5ds       Split ring with 2 join picots
r: 5ds-be-5ds-be-5ds        Ring with 2 bead element placeholders
r: 4ds-bc:3Y-4ds            Ring with 3 inline small core beads
r: 4ds-bp:YZY-4ds           Ring with beaded picot (Y-Z-Y cluster)
r: 2ds-3p-2ds-jp-2ds-3p-2ds  Mix of regular and join picots
r: 5ds-p-2rds-p-5ds         Mix of DS and reinforced DS
r: 5ss-p-5ss                Ring using single stitches

Tools

Pan Tool
Click and drag to move the camera view. Also activated by holding Space or middle-click drag.
Select Tool
Click to select elements. Drag to move. Shift+Click for multi-select. Drag empty space for box selection.
Ring Tool (R)
Adds a new ring at the center of your viewport. Use the notation field to define stitches and picots. Toggle circle / teardrop shape in the property bar.
Split Ring (SR)
A split ring has two separate halves — Side A and Side B — each with their own notation. Lets you define different stitch sequences on each arc.
Side A: sr: 5ds-p-5ds
Side B: 3ds-jp-3ds  (no sr: prefix)
The split line between the two halves is shown as a dashed divider. Picots on either side work exactly like on regular rings.
Chain Tool (C)
Adds a curved chain. Drag the bezier handles (in Path Edit mode) to shape the curve. Picots sit on the outside of the curve.
Line Tool (L)
Creates a visual connector line — no stitches, not included in pattern output. Use it to link elements, mark thread paths, or add diagram annotations.
  • Adjust thickness with the Line Width slider in the property bar.
  • Use Path Edit (PE) to curve the line just like a chain.
  • Lines can carry beads — add bead notation in the line's notation field.
Path Edit (PE)
Select a chain or line, then click PE to edit its curve. Drag the green start/end handles or the blue control handles to reshape. Handles stay fixed screen size regardless of zoom.
Picot Join (PJ)
Used to connect joint picots (jp) between elements.
  • Green dot = unconnected jp. Yellow dot = already connected.
  • In PJ mode, drag a box to select the picots you want to link, then click Link (🔗).
  • Click Unlink to remove a connection.
Image Tool
Upload a reference image to trace over. Adjust opacity, scale, position, and rotation in the image controls panel.

Tips & Tricks

  • Radial patterns: Create one petal, press D to duplicate in place, type an angle (e.g. 45° for 8-petal, 30° for 12-point). Repeat D + Enter until complete.
  • Quick color edit: Double-click any color swatch in the palette to open the color picker.
  • Rotation handles: Hold Shift (desktop) or click rotation button (mobile) to show handles. Release/click again to hide for easier movement.
  • Fast rotation: Use ↶ and ↷ buttons for instant 90° rotations instead of typing angles.
  • Symmetry with flipping: Duplicate elements and use Flip H or Flip V to create perfectly mirrored patterns.
  • Alignment: Use Arrange menu alignment options to quickly organise multiple elements (requires 2+ selected).
  • Joint picot connections: Joint picots appear as green dots on paths — yellow once connected. Click them directly to select.
  • Gradient search: Use the search bar in the gradient picker to find gradients by name or ID.
  • Auto-save: Your work is automatically saved every 3 minutes to browser storage.
  • Undo safety: Undo/Redo tracks up to 50 states — experiment freely.
  • Copy preserves: Copy/paste maintains all element properties including groups and connections.
  • Zoom tricks: Use + / - keys or mouse wheel to zoom. Press F to fit all elements in view.
  • Multi-select: Hold Shift while clicking to add elements to your selection.
  • Box selection: Drag in empty space to create a selection box around multiple elements.
  • Group workflow: Group related elements (Ctrl+G) to rotate and move them as one unit. Ungroup with Shift+G.

Customising Indicator Colors

The snap points, joint picot dots, path-edit handles and connection lines all use colors defined in a tatting-theme.json file.

To customise:

  1. Open tatting-theme.json in any text editor.
  2. Change any hex color value (e.g. "#FFA500" → "#00BFFF").
  3. Save the file and reload the app — colors apply automatically.

You can also load a theme mid-session via Options → Load Theme… without restarting.

Bead Notation

Beads are added inline in the notation string. Three size keys: Y (small), Z (medium), V (large). Sizes are configurable in Options → Bead Sizes.

bc:Y Core Bead

A single bead threaded directly onto the core thread — sits inline on the path with the thread looping behind it. Uses one ds-position.

r: 5ds-bc:Y-5ds
c: 3ds-bc:Z-3ds-bc:Y-3ds

Single size letter only (Y, Z, or V). Can appear multiple times.

bcp:Y:ZZ Core + Picot Bead

A core bead on the thread plus a branch of picot beads from its edge. Two parts separated by : — first is the core size, second is the picot sequence.

r: 4ds-bcp:Y:ZZ-4ds ← Y core, ZZ picot branch
r: 4ds-bcp:Z:YZY-4ds ← Z core, YZY branch
bp:YZY Beaded Picot

Replaces a regular picot with beads threaded onto the picot loop. Beads are arranged along a branch perpendicular to the path.

r: 5ds-bp:Y-5ds ← single bead picot
r: 5ds-bp:YZY-5ds ← three bead branch
r: 3ds-bp:2Y-3ds ← shorthand: 2 Y beads
sb:ZY Suspended Bead

Beads stacked in a straight column perpendicular to the path, like a charm dangle. A line is drawn from the path to the tip of the outermost bead.

c: 4ds-sb:Y-4ds ← single suspended bead
c: 4ds-sb:ZY-4ds ← Z closest to path, Y at tip
bjp:YZ Beaded Joint Picot

Beads clustered at the midpoint of a picot connection — shown where two linked jp picots meet. 1 bead = centred. 2 = side by side. 3+ = circular cluster.

r: 5ds-bjp:Y-5ds ← 1 bead at join
r: 5ds-bjp:3Z-5ds ← 3 Z beads clustered

Bead Sequence Syntax

Each letter = one bead of that size. A leading number repeats it: 3Y = YYY. Mix sizes freely.

Y → one small bead
YZY → small · medium · small
3Y → three small beads
2YZ → two small + one medium
V → one large bead

Polar Grids

A polar grid is a set of concentric circles with evenly-spaced "spoke" snap points on each circle. It helps you place elements at exact distances and angles — perfect for rosettes, doily rounds, and any repeating radial pattern.

Step 1 — Open the Polar Grids panel

Click the Polar Grid button in the right-side toolbar (looks like concentric circles 🔵), or find it in the Arrange menu → Polar Grid. A panel opens with a grid list on the left and the editor on the right.

Step 2 — Create a grid

Click + Add Grid. A new grid appears in the list with a default name. Click the name field at the top of the editor and type something meaningful — e.g. "Round 1" or "Main rosette".

Step 3 — Position the center

The Center X / Y fields set where the middle of the grid sits on your canvas. For most patterns, leave it at 0, 0 (the canvas origin). Click the ⌖ button to reset it back to origin at any time.

You can type any number directly — useful if you want a secondary grid centred on a specific element.

Step 4 — Add rings

Click + Add Ring. Each ring has two settings:

r Radius — how far this circle is from the center, in canvas units. The panel shows the approximate real-world size in mm next to it (based on your thread settings). Start with something like 100 for a first ring, then 200 for a second.
÷ Divisions — how many equally-spaced snap points sit on this ring. 6 gives you a hexagon, 8 an octagon, 12 a clock face. The spokes and dots update live on the canvas as you type.

Add as many rings as your pattern needs — one per round in a doily, for example.

Step 5 — Adjust color and opacity

In the header of the editor, the color swatch (circle) lets you pick a grid color — handy if you have multiple grids and want to tell them apart at a glance. The opacity slider next to it controls how faint or vivid the grid lines appear. Slide it down if the grid is distracting while you work.

Step 6 — Snap and visibility per ring

Each ring row has two small toggle buttons on the right:

 Visible — shows or hides this specific ring's circle and dots on the canvas. Hidden rings still snap if snap is on.
 Snap — when active (blue), elements will snap to this ring's dots when you drag them nearby. Turn it off for rings you only want as a visual guide.

The Visible checkbox at the top of the editor hides or shows the entire grid at once.

Angular offset (optional)

The Offset° field rotates all spokes of the grid by that many degrees. Useful when your pattern's first element doesn't sit at 12 o'clock. Each individual ring also has its own ⟳° offset that is added on top of the global one — so you can stagger rings relative to each other (e.g. offset ring 2 by 15° so its snap points fall between ring 1's).

Assigning a grid to elements (Polar Grid pivot)

Once you've created grids, you can tell each element which grid to use as its rotation and flip center. Select an element, open the properties bar, and look for the Polar Grid dropdown — pick your grid from the list.

When a grid is assigned, rotating or flipping that element will pivot around the grid's center instead of the element's own center. This is what lets you duplicate a motif and rotate it into perfect radial position.

Typical workflow: Design one repeat → assign the grid → duplicate in place (D) → type a rotation angle → press Enter → repeat until the circle is complete.

Quick tips

  • ✓Grids are saved globally — they persist across projects. You don't need to recreate them every time.
  • ✓You can have multiple grids open at once — e.g. one for the inner round and one for the outer round of a doily.
  • ✓The mm size hint next to each radius updates if you change thread settings in Thread Properties — use it to match a real-world pattern size.
  • ✓If snap feels "too sticky", lower the number of divisions on a ring or turn snap off for rings you're not actively working with.
  • ✓Press Escape or click outside the panel to close it without losing any settings.
