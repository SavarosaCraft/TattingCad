Menu Bar

Four menus at the top of the screen: File, Arrange, Options, and Help. All menus are scrollable if the window is short.

📁 File
Project saving, loading, and export.
Project
🆕
New Ctrl+N
Clears the canvas to start fresh. If the canvas has content a confirmation dialog appears first to prevent accidental data loss.
💾

Opens the Save dialog to name the project, then saves a .json file. Generates a prveiw for the loader.
📂
Load Project
Opens the Recent Projects browser showing SVG thumbnail previews of previously saved or loaded files. Use Browse for file… to pick any .json from disk. If the canvas has work a confirmation step appears first.
📂
Browse for File...
System window to pick any .json from disk. If the canvas has work a confirmation step appears first.
Export
↗
Export SVG
Saves the pattern or Schematic View as a scalable vector file. Open it in any browser, Inkscape, or Illustrator to view, print, or convert to other formats at any size.
📄

Generates written pattern text onto the clipboard. Elements with order number will be listed in order. Uses thread properties to estimate needed thread length. Also if pattern has any beads it will summarize them by type.

Order #

Rotatie 90 Degree CCW

Rotation angle

Available Editing Modes

Picot Edit Mode

Entered via left toolbar: Joint Picot Tool, or Shift+J. Exit via 'Exit mode' button or ESC.
In this mode you can select and connect elements that have the joint picot notation.
JP - joined picots, JPG - joined picots with guide point, or BE beaded element, with joined picot setting.
CJ - core join, CJP - core joined picot also selectable and joinable in this mode.

Using join points
  • Circle markers indicate where joining points is possible.
  • Click to select, or drag to multi-select. Purple colors indicate selected points.
  • Yellow circles and dashed lines indicate existing connections.
  • If the selected elements have different materials (colors), then the first selected picot will determine the color of the joints.
Mode Banner Row
JOIN
After selecting more than one join picots, this button will connect them together.
Shortcut: J
CUT
Selecting already connected picots, this button will remove their existing connections.
Shortcut: C or DELETE
✕ Exit Mode
Exits Mode and returns to the normal property bar.
Shortcut: ESC

Bead Edit Mode — Assigning Beads to Picots

Assign beads to any element containing the BE (Beaded Element) notation. Exception is the Line element, that handles its own beads in the property bar.
Enter with Shift+B or the 'Beading mode" on left toolbar. Exit via 'Exit mode' button or ESC.

Selecting bead positions
  • Diamond markers indicate BE (Beaded Element) notations.
  • Click to select, or drag to multi-select.
  • Edits apply to all selected positions.

Bead structures

Core only
A bead threaded onto the core thread at this picot. No picot loop.
Core + plain picot
Core bead(s) plus a standard un-beaded picot loop above.
Core + beaded picot
Core bead(s) plus a picot loop that itself carries beads.
Suspended
Beads string hang from the pico, held up by a closing bead.
Beaded picot
Only the picots has beads.

Connectable join point TOGGLE

A toggle to let a BE (Beaded Element) behave like a Join picot. In Join Picot mode these will be selectable, and can nonnect to one or more other joined picots.
Copy / Cut / Paste
  • Behaves as the Clipboard, just for the Beaded elements.

Dropdowns

Select the desired beads to apply to the selected elements.
Edit the available beads using the 'Library...' button.

Tatting Order Mode

Entered via View → Tatting Order… Exit via 'Exit mode' button or ESC.
Dragging, multi-select, and notation editing are all disabled — click elements to select them one at a time and assign sequence numbers.

Mode Banner Row
Round dropdown
Assign "rounds" to elements. Placing them in a separate group makes organizing repetitive patterns easier. Editing the names of rows or deleting them is possible within the dropdown. When an element assigned to a group, it will change the color of the order number.
Progress counter
Shows {numbered} / {total} numbered — updates live as you work. Counts all elements including lines (lines can carry order numbers too).
Hint text
Reminds you to select an element to assign its number. Disappears once an element is selected.
✕ Exit Mode
Exits Tatting Order Mode and returns to the normal property bar.
Number Assignment Row (appears when an element is selected)
Order # input
Pre-filled with the element's current number if it already has one. Type any positive integer and press Enter or click the ↵ button to assign. Cleared automatically when you click a different element.
↵ (submit)
Assigns the typed number to the selected element. Disabled while the field is empty or non-numeric. If the number is already taken by another element, the Conflict popup appears.
Assign Next
One-click shortcut that assigns the next available gap number — the lowest positive integer not yet used by any element. The button label shows the number that will be assigned so you can confirm before clicking.
Assign to Group
Assigns the selected element to the selected Group.
Clear
Removes the order number from the selected element entirely. Disabled if the element has no number.
Current # badge
Small gold #N badge shown next to Clear when the element already has a number — quick confirmation of what's assigned.
Canvas behaviour in this mode
Pink glow
All elements without an order number glow pink at all times while the mode is active. Glow disappears as soon as a number is assigned.
Gold / Group colored numbers
Assigned order numbers appear as colored labels at the midpoint of each element — rings at their center, chains and lines at the arc midpoint. Ungrouped elements show in gold. Elements assigned to a round use that round's color.
No dragging
Elements cannot be moved while this mode is active. Clicks select; nothing drags. Exit the mode first to reposition elements.
Gray R badge
Elements marked as Repeat show a gray R badge instead of a number. They are treated as assigned (no pink glow, not counted as unnumbered) but do not appear in the pattern output. Use this for motifs you work multiple times without needing a separate order number — for example, repeating rings 1–6 four times.
Keyboard Shortcuts (element selected, cursor not in a text field)
N
Assigns the next available number to the selected element — same as clicking the Assign Next button.
R
Marks the selected element as Repeat. Shows a gray R badge; element is excluded from pattern output but treated as assigned.
Delete / Backspace
Clears the order assignment from the selected element (number or repeat mark). Same as clicking the Clear button.
Conflict Popup (amber border)
Appears when you try to assign a number that is already in use by another element.
Swap
The selected element gets the entered number; the element that previously held it gets the selected element's old number. Grayed out if the selected element has no existing number — there is nothing to swap to.
Shift all higher numbers up
The selected element gets the entered number. Every element with a number equal to or higher than the entered number is incremented by one to make room. Gaps in the existing sequence are preserved.
Cancel
Dismisses the popup. Nothing changes — the selected element keeps its previous number (or stays unnumbered).
SVG Export
Order Numbers layer
When any element has an order number, SVG export adds a dedicated Order Numbers layer (Inkscape/Illustrator compatible). Each number is a gold text element with a black outline, positioned at the element midpoint. The layer can be hidden or deleted in a vector editor without affecting the pattern artwork.

Visual grid of recently saved or loaded projects with SVG thumbnail previews. Thumbnails are generated automatically at save/load time. Up to 20 entries are kept.

Triggered by File → New when the canvas has content.

Warns that unsaved work will be lost. Confirm to clear the canvas, or cancel to go back.

🔵 Polar Array Dialog
Triggered by Arrange → Array → Polar Array…
Count
Total copies including the original. Quick-picks: 4, 6, 8, 12.
Fill angle
Arc to fill in degrees. 360° = full circle. Copies are spaced evenly across this angle.
Pivot
Rotation center — a specific polar grid, or the center of the selection itself.
Peek
Hold to fade the dialog and see the live blue ghost preview on the canvas clearly.
Apply
Creates all copies as a single undoable step.
➡ Linear Array Dialog
Triggered by Arrange → Array → Linear Array…
Count
Total copies including the original. Quick-picks: 3, 4, 6, 8.
Direction
Angle in degrees for the array direction. H (0°) and V (90°) presets, or type a custom angle.
Spacing
Distance between each copy in canvas units.
Rotation/step
Extra rotation added to each copy cumulatively. 0 = none. 15 = each copy 15° more than the previous. Reset button clears to 0.
Summary line
Plain-language description of the current settings updated live.
Peek
Hold to fade the dialog and see the live green ghost preview on the canvas clearly.
Apply
Creates all copies as a single undoable step.
🌀 Spiral Array Dialog
Triggered by Arrange → Array → Spiral Array…
Count
Total copies including the original. Quick-picks: 4, 6, 8, 12.
Type
Archimedean — uniform gap between turns. Geometric — gap grows by a multiplier each step.
Gap
(Archimedean) Distance added to the radius per step in canvas units.
Growth factor
(Geometric) Multiplier applied to the radius each step. 1.2 = grows 20% per copy.
Rotate elements
When checked, each copy is rotated to face outward along the spiral. Uncheck to keep the original orientation.
Peek
Hold to fade the dialog and see the live pink ghost preview on the canvas clearly.
Apply
Creates all copies as a single undoable step.
